Unmanned Aerial Sensing Vehicles Details

Project information

  • Technology: Python
  • Project date: Fall 2018 - Spring 2019
  • Project overview: Powerpoint
  • Autonomous Flight: Video

Build a Hobbypower F550 Hexacopter drone and program it using a Raspberry Pi and a Pixhawk 4 flight controller. Mount and program a gas sensor, optical sensors, particle sensors, humidity, and radiation sensors. Then transmit data from the sensors mounted on the drone to an onsite computer.
